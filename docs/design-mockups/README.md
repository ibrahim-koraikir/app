# Design Mockups

This folder contains HTML/Tailwind CSS design mockups that were used as reference for implementing the Jetpack Compose UI.

These files are **not** loaded at runtime by the Android app. They serve as design artifacts only.

## Files

- `welcome.html` - Onboarding/welcome screen design
- `home.html` - Home screen with website categories
- `tabs.html` - Tab management screen
- `settings.html` - Settings screen
- `permissions.html` - Permission request screen
- `features.html` - Feature showcase screen

## Translation Guide

See `.kiro/specs/entertainment-browser/design-translation-guide.md` for details on how these designs map to Compose components.
